# Semantic Search using Vector Embeddings

## 📌 Project Overview
This project implements a simple semantic search system using vector embeddings. It allows users to input a query and retrieves the most similar sentence based on meaning.

## 🚀 Features
- Converts text into embeddings using Sentence Transformers
- Compares query with dataset
- Returns the most relevant result

## 🛠️ Tech Stack
- Python
- sentence-transformers

## ▶️ How to Run

1. Install dependencies:
pip install sentence-transformers

2. Run the program:
python main.py
